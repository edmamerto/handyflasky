# Restful Flask
> I use this for my infrastructure testing

##  Setup
```sh
$ pip install virtualenv
$ cd path/to/proj
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```
## Run
```sh
$ python app.py
```
Expected Output
```
Kamusta, Mundo!
```
